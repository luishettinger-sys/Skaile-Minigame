# Waffen-Sounds (optional, CC0-Samples)

Lege hier echte Schuss-Samples ab. Das Spiel lädt sie automatisch beim Start.
Ist eine Datei nicht vorhanden, spielt weiterhin der synthetische Sound.

## Dateinamen (genau so, einer pro Sound-Typ)

| Datei         | benutzt von Waffen (Beispiele)                  |
|---------------|-------------------------------------------------|
| `blaster.*`   | Quack-Blaster, Fork-Trishot, Glitch-Cannon …    |
| `shotgun.*`   | Bread-Shotgun, Exception-Flak, Supernova        |
| `smg.*`       | Stack-SMG, Lightning-Arc, Bit-Needler           |
| `rail.*`      | Nullpointer-Rail                                |
| `cannon.*`    | Heap-Kanone, Sawblade, Void-Lober, Recursion    |
| `minigun.*`   | Loop-Minigun                                     |
| `sniper.*`    | Segfault-Sniper, Photon-Beam                     |
| `pulse.*`     | Async-Pulswelle                                  |

Endung `.mp3`, `.wav` oder `.ogg` (in dieser Reihenfolge probiert).

## Worauf achten
- **Kurz** (~0,1–0,8 s), **trocken** (wenig Hall), sauber geschnitten am Anfang.
- Lautstärken zwischen den Dateien grob angleichen.
- **Lizenz: CC0 / public domain** (z. B. Freesound.org mit CC0-Filter,
  oder das gratis **Sonniss GDC Game Audio Bundle**).

## Quellen-Suchbegriffe
`pistol shot`, `shotgun blast`, `smg burst`, `railgun`, `cannon / launcher`,
`minigun`, `sniper rifle shot`, `energy pulse`.
